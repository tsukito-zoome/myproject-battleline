# バトルライン (Battle Line)

ボードゲーム「バトルライン」のブラウザ実装です。

🎮 **プレイする**: [battle-line-coral.varcel.app]

## 遊び方

- **AI対戦** — コンピュータと1人で対戦
- **ローカル対戦** — 同じ端末で2人対戦（画面を交互に渡す）
- **オンライン対戦** — ルームコードを共有して離れた相手と対戦（Firebase使用）（実装前）

## ゲームについて

9本のフラグを挟んで2人のプレイヤーが戦います。
各フラグに3枚のカードを並べてフォーメーションを作り、強いフォーメーションを持つ側がフラグを獲得。
**5本以上**または**連続3本**のフラグを取った方が勝利です。

フォーメーションの強さ（強い順）:
1. ウェッジ（同色の連番3枚）
2. ファランクス（同数字3枚）
3. バタリオン（同色3枚）
4. スカーミッシュ（連番3枚）
5. ホスト（その他）

戦術カード（偵察・脱走・裏切りなど）を使った高度な戦略も可能です。

## 技術スタック

- React + TypeScript (Vite)
- Firebase Realtime Database（オンライン対戦）
- Vercel（ホスティング）
